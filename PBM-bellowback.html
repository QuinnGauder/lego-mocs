Horizon: Fire Bellowback

Pieces: 1277
Dimensions (L⋅W⋅H): 39.8cm x 23.4cm x 22.9cm
Additional Links: Rebrickable Instructions

             A large slightly above minifigure scale diorama of the Fire Bellowback machine from the Horizon series. Designed to match a similar design language of the official Tallneck set from the same series. References to the game series are hidden amongst the build such as medicinal berries, ridgewood, and tall grass. The small components around the neck can be removed, in the same way they can be shot off the machine in the games, to make the machine appear destroyed/weakened.
