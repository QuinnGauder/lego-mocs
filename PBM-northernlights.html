Microscale Northern Lights

Pieces: 122-124
Dimensions (L⋅W⋅H): 6.0cm x 6.5cm x 5.8cm
Additional Links: Rebrickable Instructions

             In a remote snowy northern forest, by a frozen river, lies a secluded cabin. One night the skies clear showing the magnificent spectacle of Aurora Borealis amongst the starry night. This build was originally designed for Bricksculpt's 2025 Winter Challenge Micro Build category—for which it won first place! The competition's only requirement was for the build to be roughly within an 8x8 footprint and represent a winter scene of some kind.
